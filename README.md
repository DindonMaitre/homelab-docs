# Homelab Infrastructure

18+ services on Proxmox LXC containers.

**Hardware**: Chuwi Ubox (32GB), Synology DS216SE (2TB), Raspberry Pi 4  
**Services**: Jellyfin, Radarr, Sonarr, Navidrome, Nginx Proxy Manager, etc.  
**Domain**: maitredindon.online (Cloudflare + SSL)

## Services

- Media Stack (Jellyfin, Radarr, Sonarr, Lidarr, Prowlarr)
- Music (Navidrome, slskd, Soularr)
- Infrastructure (Nginx Proxy Manager, Glance)
- Other (Crafty, ezBookkeeping, MySpeed)

Full documentation available in Notion.
